MOROway App

History

MOROway App started as a small Adobe Flash animation project 2011 under the name "MOROway Animation" (Versions 1.0-3.2). Four years later the next and last update for the Flash version was created featuring e.g. a third train (Version 4.0). Since 2018 MOROway App is a modern HTML5/JavaScript web app (Version 5.0). In the period after this major update several new features like switches for the trains and an automatic mode for the cars were added. Since 2019 a multiplayer mode is available (Version 6.0). The following releases improved usability and performance.
Since 2014 MOROway App is also available in Google's Play Store. Microsoft Store, F-Droid and Ubuntu's Snap Store followed.
MOROway App is an open source project and you may find the source code on Github (starting from April 2018). From 2022 on translations are handled using Weblate.
